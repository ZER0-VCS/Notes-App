## [0.2.0] - 2025-11-15

### Добавлено
- **Обработка ошибок**: Полная обработка исключений в `notes.py` и `gui.py`
- **Логирование**: Система логирования с записью в файл `notes_app.log`
- **Валидация данных**: Метод `Note.validate()` для проверки корректности данных
- **Горячие клавиши**:
  - `Ctrl+S` - Сохранить заметку
  - `Ctrl+N` - Создать новую заметку
  - `Ctrl+D` - Удалить заметку
- **Резервное копирование**: Метод `NoteStore.create_backup()` с хранением последних 10 копий
- **Атомарная запись**: Сохранение через временный файл для предотвращения повреждения данных

### Изменено
- Все вызовы `print()` заменены на `logging`
- Улучшены сообщения об ошибках в GUI
- Обновлен `.gitignore` для исключения логов и резервных копий

### Исправлено
- Обработка повреждённых JSON файлов с автоматическим созданием резервной копии
- Корректная обработка ошибок при чтении/записи файлов

## [0.1.0] - 2025-11-12

### Добавлено
- **Базовая функциональность (MVP)**:
  - Класс `Note` для представления заметки
  - Класс `NoteStore` для управления коллекцией заметок
  - CRUD операции (создание, чтение, обновление, удаление)
  - Хранение в JSON формате (`~/.notes_app/notes.json`)
  
- **Графический интерфейс (PySide6)**:
  - Список заметок с сортировкой по дате
  - Редактор с полями для заголовка и текста
  - Кнопки: Создать, Сохранить, Удалить
  - Отслеживание несохранённых изменений
  - Подтверждение при удалении и закрытии с несохранёнными изменениями
  
- **Документация**:
  - `README.md` - Основная документация
  - `QUICKSTART.md` - Руководство по быстрому старту
  - `IMPROVEMENTS.md` - Рекомендации по улучшению
  - `TODO.md` - Список задач

### Технические характеристики
- Python 3.13
- PySide6 для GUI
- Модульная архитектура
- Поддержка Unicode (UTF-8)
- Версионирование заметок
