# Руководство по тестированию версии 0.5.0

## Автоматические тесты

Запустите автоматический тест для проверки всех функций:

```bash
python test_v050_features.py
```

**Результат:** Все 6 тестов должны пройти успешно.

## Ручное тестирование

### 1. Тестирование GUI выбора темы

1. Запустите приложение: `python main.py`
2. Откройте меню **Вид** → **Тема**
3. Выберите тему (Светлая/Темная/Голубая/Зеленая)
4. **Результат:** Тема применяется сразу без перезапуска
5. Закройте и откройте приложение снова
6. **Результат:** Выбранная тема загружается автоматически

### 2. Тестирование диалога настроек

1. Откройте **Файл** → **Настройки**
2. Протестируйте вкладку **Общие**:
   - Измените интервал автосохранения (3-10 сек)
   - Измените интервал автосинхронизации (30-300 сек)
3. Протестируйте вкладку **Редактор**:
   - Выберите другой шрифт
   - Измените размер шрифта
4. Протестируйте вкладку **Оформление**:
   - Выберите другую тему
5. Нажмите **Применить**
6. **Результат:** Все изменения применяются мгновенно:
   - Шрифт меняется сразу
   - Тема применяется без перезапуска
   - Интервалы обновляются

### 3. Тестирование экспорта заметок

#### Экспорт одной заметки

1. Выберите заметку из списка
2. Откройте **Файл** → **Экспорт** → **Экспортировать текущую заметку**
3. Выберите формат:
   - **Markdown** - проверьте front-matter (title, tags, created, id)
   - **Plain Text** - проверьте заголовок и текст
   - **HTML** - откройте в браузере, проверьте форматирование

#### Массовый экспорт

1. Откройте **Файл** → **Экспорт** → **Экспортировать все заметки**
2. Выберите формат (Markdown/TXT/HTML)
3. **Результат:** Создается ZIP архив со всеми заметками
4. Распакуйте архив и проверьте содержимое

### 4. Тестирование горячего применения

1. Откройте диалог настроек
2. Измените тему с темной на светлую
3. Нажмите **Применить** (не закрывая диалог)
4. **Результат:** Тема меняется сразу, диалог остается открытым
5. Измените шрифт на другой
6. Нажмите **Применить**
7. **Результат:** Шрифт меняется мгновенно
8. Измените интервал автосохранения
9. Создайте/измените заметку
10. **Результат:** Автосохранение работает с новым интервалом

### 5. Проверка сохранения настроек

1. Измените все настройки в диалоге
2. Нажмите **ОК**
3. Закройте приложение
4. Откройте снова
5. **Результат:** Все настройки загружены (тема, шрифт, интервалы)

## Известные ограничения

- Некоторые inline стили кнопок не изменяются при смене темы (требуется полная перезагрузка для идеального результата)
- Цвета выделения при поиске не изменяются динамически
