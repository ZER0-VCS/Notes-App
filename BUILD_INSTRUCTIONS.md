# Инструкция по сборке EXE файла

## Сборка выполнена успешно!

Исполняемый файл приложения находится в папке `dist/NotesApp.exe`

## Как пользоваться

1. Просто запустите файл `NotesApp.exe` из папки `dist/`
2. Приложение запустится без необходимости установки Python или других зависимостей
3. Все данные заметок сохраняются в папке пользователя

## Распространение

Для распространения приложения достаточно скопировать только файл `NotesApp.exe` - это полностью автономный исполняемый файл.

## Повторная сборка

Если нужно пересобрать EXE файл после изменений в коде:

```bash
python -m PyInstaller notes_app.spec --clean
```

## Настройки сборки

Все настройки сборки находятся в файле `notes_app.spec`:
- Имя итогового файла: `NotesApp`
- Режим: без консоли (GUI приложение)
- Тип сборки: onefile (один исполняемый файл)
- Включены все необходимые модули PySide6

## Размер файла

Исполняемый файл содержит:
- Интерпретатор Python
- Все библиотеки PySide6 (Qt)
- Исходный код приложения

Поэтому его размер составляет около 50-100 МБ, что является нормальным для GUI приложений на Qt.
