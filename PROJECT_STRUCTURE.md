# Структура проекта Notes-App

## Корневая директория

```
Notes-App/
├── src/                    # Исходный код приложения
│   ├── __init__.py        # Инициализация пакета
│   ├── gui.py             # Графический интерфейс (PySide6)
│   ├── notes.py           # Модель данных (Note, NoteStore)
│   ├── sync.py            # Синхронизация между устройствами
│   ├── themes.py          # Система управления темами
│   ├── export.py          # Экспорт заметок
│   ├── settings_dialog.py # Диалог настроек
│   └── example_themes.py  # Примеры использования тем
│
├── docs/                   # Документация
│   ├── CHANGELOG.md       # История изменений
│   ├── ROADMAP.md         # План развития
│   ├── CONTRIBUTING.md    # Руководство для контрибьюторов
│   ├── SYNC_IMPROVEMENTS.md   # Документация улучшений синхронизации
│   ├── TESTING_v050.md    # Руководство по тестированию v0.5.0
│   ├── TEST_REPORT_v050.md    # Отчет о тестировании v0.5.0
│   └── Отчёт по созданию.ini  # Отчет о создании проекта
│
├── tests/                  # Тесты
│   ├── test_v050_features.py  # Тесты функций v0.5.0
│   ├── test_fixes_v050.py     # Тесты исправлений v0.5.0
│   ├── test_search.py     # Тесты поиска
│   └── ... (другие тесты)
│
├── main.py                # Точка входа в приложение
├── requirements.txt       # Зависимости Python
├── README.md              # Основная документация
├── .gitignore             # Игнорируемые Git файлы
└── notes_app.log          # Лог-файл приложения
```

## Описание модулей

### src/

**gui.py** - Главный модуль графического интерфейса
- Класс `NotesApp` (QMainWindow)
- Управление окнами и виджетами
- Обработка событий пользователя
- Интеграция с синхронизацией

**notes.py** - Модель данных
- Класс `Note` - структура заметки
- Класс `NoteStore` - хранилище заметок
- CRUD операции
- Сохранение/загрузка JSON

**sync.py** - Синхронизация
- Класс `SyncManager` - менеджер синхронизации
- Алгоритм Last-Writer-Wins (LWW)
- Обработка конфликтов
- Tombstones для удаления

**themes.py** - Темы оформления
- Класс `Theme` - определение темы
- Класс `ThemeManager` - управление темами
- 4 предустановленные темы
- Система применения стилей

**export.py** - Экспорт данных
- Класс `NoteExporter` - экспортер заметок
- Форматы: Markdown, TXT, HTML
- Массовый экспорт в ZIP
- Генерация безопасных имен файлов

**settings_dialog.py** - Настройки
- Класс `SettingsDialog` - диалог настроек
- 3 вкладки: Общие, Редактор, Оформление
- Сохранение конфигурации
- Горячее применение изменений

### docs/

Вся документация проекта:
- История изменений (CHANGELOG)
- План развития (ROADMAP)
- Руководства по использованию
- Отчеты о тестировании

### tests/

Автоматические и ручные тесты:
- Функциональные тесты
- Тесты исправлений
- Интеграционные тесты

## Преимущества новой структуры

✅ **Организация** - чистое разделение кода, документации и тестов
✅ **Масштабируемость** - легко добавлять новые модули
✅ **Импорты** - использование пакетов Python
✅ **Читаемость** - ясная структура для новых разработчиков
✅ **Совместимость** - обратная совместимость импортов
