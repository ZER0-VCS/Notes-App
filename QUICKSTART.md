Все необходимые компоненты установлены:

## Как запустить приложение

### Вариант 1: Через активацию окружения

```powershell

.\venv\Scripts\Activate.ps1

# 2. Запустите приложение
python main.py
```

### Вариант 2: Прямой запуск (без активации)

```powershell
C:/Users/ilyak/OneDrive/Desktop/Notes-App/.venv/Scripts/python.exe main.py
```

## Основные функции

### 1. Создание заметки
- Нажмите **"➕ Создать заметку"**
- Введите заголовок и текст
- Нажмите **"💾 Сохранить"**

### 2. Редактирование заметки
- Выберите заметку из списка слева
- Измените заголовок или текст
- Кнопка "💾 Сохранить" станет активной
- Нажмите для сохранения изменений

### 3. Удаление заметки
- Выберите заметку из списка
- Нажмите **"🗑️ Удалить"**
- Подтвердите удаление

### 4. Автоматическое отслеживание изменений
- Приложение отслеживает все несохраненные изменения
- При переключении между заметками появится запрос сохранения
- При закрытии приложения будет предложено сохранить изменения

## Устранение неполадок

### Ошибка: "PySide6 не найден"
```powershell
# Переустановите зависимости
pip install -r requirements.txt
```

### Ошибка: "Не удается выполнить скрипт"
```powershell
# Измените политику выполнения
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### Приложение не запускается
```powershell
# Проверьте версию Python (должна быть 3.13)
python --version

# Проверьте установку PySide6
pip show PySide6
```
